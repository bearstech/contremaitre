Contremaître
============

Build a _supervisord_ conf from a [Procfile](https://devcenter.heroku.com/articles/procfile).

Contremaitre is mix from foreman and gaffer.

Test it
-------

Be polite, make a virtualenv

    virtualenv .
    source bin/activate
    pip install -r requirements.txt

Launch test

    ./bin/nosetests
